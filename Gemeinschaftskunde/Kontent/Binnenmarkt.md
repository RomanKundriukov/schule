## **Binnenmarkt** -> Bereich, indem Staaten ohne Zölle oder Beschränkungen handeln können.

>**Europäische Binnenmarkt**
>	**Freien Warenverkehr**
>	**Freier Personenverkehr**
>	**Freie Kapitalverkehr z.B. Bankkonto oder Immobilien kaufen**
>	**Freier Dienstleistungsverkehr z.B. Handverkehr**


| **Vorteile**                                          | **Nachteile**                           |
| ----------------------------------------------------- | --------------------------------------- |
| weniges Kosten                                        | Schmuggel / handelt mit illegale Waren  |
| weniges / keine Kontrolle                             | Arbeitsplätze können verloren gehen     |
| mehr Gewinn für Unternehmen                           | Produktionsverlagerung in andere Länder |
| weniger Bürokratie (= Regeln, Vorschriften)           | für kleine Betriebe mehr Konkurenz      |
| einfaches Reisen                                      |                                         |
| mehr Auswahl / Möglichkeiten für Firmen + Verbrauches |                                         |
