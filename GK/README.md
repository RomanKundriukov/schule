- Parlament + (BT) wählt den Regierungschef (BK)
- Regierung hat die Mehrheit
    - Verabschiedet Gesetzte
    - Fraktion (= Partei im BT) muss geschlossen abstimmen / Fraktionsdisziplin

#### Welche Aufgaben erfüllt eine Wahl?

1. Herrschaftskontrolle = wähler entscheiden, ob Partei regieren soll.

2. Herrschaftslegitimation = durch eine wahl erhält der wahlsieger die Mehrheit, d.h. das Recht zu regieren.

3. Delegation = wähler entscheided sich für eine Partei, die ihre Interessen vertritt.

4. Polit Teilnahme = durch wahlen kann man mitentscheiden

## Deutschland: Direkte und indirekte Demokratie

#### Indirekte Demokratie:
- In Deutschland dominiert die repräsentative Demokratie. Das Volk wählt Abgeordnete (z. B. im Bundestag), die stellvertretend Entscheidungen treffen. Dies gilt für die meisten politischen Prozesse.

#### Direkte Demokratie:
- Elemente der direkten Demokratie sind begrenzt. Sie kommen auf Länderebene (z. B. Volksbegehren, Volksentscheide) oder bei kommunalen Entscheidungen vor. Auf Bundesebene sind Volksentscheide selten (z. B. Wiedervereinigung, EU-Beitritt).
